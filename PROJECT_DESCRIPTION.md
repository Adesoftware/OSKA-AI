# OSKA AI Demo Script

Good day Judges.

My project is OSKA AI, which stands for Offline Smart Knowledge Assistant.

OSKA AI is an enterprise productivity assistant that operates locally without requiring cloud-based AI services after installation.

Today I will demonstrate:

1. Business Letter Generation
2. Memo Creation
3. Text Summarization
4. General AI Assistance

The solution was developed using Python and Ollama and is capable of running on modest hardware.

This project demonstrates how organizations can leverage artificial intelligence while maintaining control over sensitive information and reducing dependence on internet connectivity.

Thank you.
