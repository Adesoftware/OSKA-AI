# OSKA AI

**Offline Smart Knowledge Assistant**

**Tagline:** Enterprise Intelligence Without the Cloud

## Overview

OSKA AI is an offline enterprise assistant built with Python and Ollama. It helps users generate business letters, draft memos, summarize text, and answer questions without requiring cloud services.

## Features

* Offline AI Chat
* Business Letter Generation
* Memo Drafting
* Text Summarization
* Lightweight Local AI Processing
* No Internet Required After Setup

## Technology Stack

* Python 3.13
* Ollama
* Requests Library
* Visual Studio Code
* Git & GitHub
